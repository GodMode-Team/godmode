Create a clean HTML dashboard called "Morning Overview" that shows:

- Today's date and a greeting
- Top 3 priorities for the day (from tasks)
- Today's calendar events (next 5)
- Active queue items and their status
- A focus score or readiness rating

Style: clean, minimal, dark-friendly. Use CSS grid. No external dependencies.
