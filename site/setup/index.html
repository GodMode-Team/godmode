Private Setup Guide

Install GodMode on your VPS

One command installs everything. Follow the steps below to get running in under 5 minutes.

1 Run the installer

SSH into your server and paste this:

curl -fsSL https://lifeongodmode.com/install.sh | sh

This automatically installs Node.js, the OpenClaw runtime, and the GodMode plugin. It detects your VPS environment and configures everything for remote access.

What gets installed

  • Node.js 22 (if not already present)
  • OpenClaw CLI (the AI gateway runtime)
  • GodMode plugin (the AI operating system)
  • Security token for the gateway

2 Connect an AI model

GodMode needs an AI brain to think with. Pick one:

Option A: Anthropic API Key Recommended

Best quality. Pay-per-use. Get a key at console.anthropic.com

echo 'export ANTHROPIC_API_KEY="sk-ant-YOUR-KEY-HERE"' >> ~/.bashrc
source ~/.bashrc

Option B: OpenAI / ChatGPT subscription

Uses your existing ChatGPT Plus ($20/mo) or Pro ($200/mo) subscription.

openclaw models auth login

Select "OpenAI" when prompted. Follow the browser link to authorize.

Option C: Other providers

Any of these work — add the key to ~/.bashrc and source ~/.bashrc:

  • GEMINI_API_KEY — Google Gemini (get key)
  • OPENROUTER_API_KEY — OpenRouter, 100+ models (get key)
  • XAI_API_KEY — xAI Grok (get key)

3 Restart the gateway

After adding your AI key, restart so the gateway picks it up:

openclaw gateway restart

4 Access GodMode

Since your GodMode is running on a server, you need a way to reach it from your browser.

Option A: Tailscale Recommended

Tailscale creates a private network between your devices. Install it on your VPS and your laptop/phone.

# On your VPS:
curl -fsSL https://tailscale.com/install.sh | sh
sudo tailscale up

# It will print a login URL — open it to connect your VPS to your Tailscale network
# Then get your VPS's Tailscale IP:
tailscale ip -4

Then open in your browser:

http://YOUR-TAILSCALE-IP:18789/godmode

Install Tailscale on your laptop/phone too, using the same account.

Option B: SSH Tunnel Quick

From your laptop, open a tunnel to your server:

ssh -L 18789:localhost:18789 user@your-server-ip

Then open in your browser:

http://localhost:18789/godmode

The tunnel stays open as long as the SSH session is active.

5 You're in

GodMode's onboarding will walk you through the rest. It'll ask your name, set up your workspace, and introduce you to your AI ally.

Quick reference

  • Gateway status: openclaw gateway status
  • Restart gateway: openclaw gateway restart
  • Update GodMode: curl -fsSL https://lifeongodmode.com/install.sh | sh
  • Logs: tail -f ~/.openclaw/logs/gateway.log

Troubleshooting

"Gateway Offline" or "Connection lost" — If you see "origin not allowed" in the browser console, allow your Tailscale IP:

openclaw config set gateway.controlUi.allowedOrigins '["http://YOUR_TAILSCALE_IP:18789"]'
openclaw config set gateway.controlUi.allowInsecureAuth true
openclaw gateway restart

Replace YOUR_TAILSCALE_IP with the output of tailscale ip -4 on your VPS.
