Quick Fixes

Gateway not responding

If the GodMode UI shows a connection error or won't load, restart the gateway.

$ openclaw gateway restart

Plugin not loading

If GodMode doesn't appear after install, re-install the plugin and restart.

$ openclaw plugins install @godmode-team/godmode && openclaw gateway restart

Full reinstall

Nuclear option. Removes and reinstalls everything cleanly.

$ curl -fsSL https://lifeongodmode.com/install.sh | sh

Authentication issues

If Claude isn't responding, re-authenticate your AI connection.

$ openclaw auth login

Check gateway status

Verify whether the gateway is currently running and on which port.

$ openclaw gateway status

Common Questions

What does GodMode cost?

GodMode is free and open source. It uses your existing Claude subscription (Pro, Max, or API key) — there are no additional API costs for Pro/Max subscribers.

Does GodMode work on Windows?

GodMode supports macOS and Linux natively. Windows users can run it through WSL2 (Windows Subsystem for Linux).

Where is my data stored?

All data stays local on your machine. GodMode stores state in ~/godmode/ and integrates with your Obsidian vault if configured. Nothing is sent to external servers.

Can I use GodMode without Obsidian?

Yes. Obsidian integration is optional. GodMode falls back to its own local storage in ~/godmode/memory/ when no vault is configured.
