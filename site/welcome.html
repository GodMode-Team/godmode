Get Started

  1. Install OpenClaw
    npm i -g openclaw
  2. Install the GodMode plugin
    openclaw plugins install godmode
  3. Activate your license
    openclaw godmode activate YOUR-KEY
  4. Start the gateway
    openclaw gateway run
    Then open http://localhost:18789/godmode

Need Help?

Run openclaw godmode doctor to diagnose any setup issues.
