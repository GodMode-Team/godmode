Getting Started

Terminal
# Prerequisites: Claude Code, GitHub CLI (gh), Node 22+, pnpm

# 1. Clone the repo
git clone https://github.com/GodMode-Team/godmode-plugin.git
cd godmode-plugin

# 2. Install & build
pnpm install
pnpm build

# 3. Verify
gh auth status            # should show your GitHub username
pnpm typecheck           # should pass with no errors

# 4. Start working
claude                     # open Claude Code in this directory
The only things humans do

3 Rules for Humans

👁

Notice bugs

If something looks wrong, file it with /bug. Don't assume someone else saw it.

👀

Review PRs

At least one person must approve before merging. /pr-review helps you understand the code.

⚡

Don't sit on PRs

If a PR is open more than a day, review it. Stale PRs cause merge conflicts.

When things go wrong

Troubleshooting

"Claude won't let me edit files"
That's the branch guard. You're on main. Tell Claude what you want to do and it'll create a branch automatically.
"My PR CI is failing"
Run /fix on your branch, or ask Claude to "fix the typecheck errors on this branch."
"I have merge conflicts"
Run /sync and Claude will help rebase your branch onto the latest main.
"Someone is already fixing the same thing"
That's what GitHub Issues prevents. /fix checks if an issue is assigned before starting. Pick a different one.
"I need to update after someone merged a fix"
Run /sync. It pulls the latest code, installs dependencies, and rebuilds everything.
