## What GodMode Is

GodMode is a personal AI operating system — an always-on AI ally that runs locally, manages your memory, calendar, tasks, and team coordination through natural conversation. Built as a plugin on top of OpenClaw (open-source AI agent framework). Think: Jarvis for entrepreneurs and their teams.

We're pre-revenue with a working product (v1.8), a team of 4 using it daily, and first paying customers lined up. This is a "join before the rocket takes off" role.

## What You'd Own

- **The GodMode plugin codebase** — TypeScript, ~15K LOC, Lit web components UI
- **Bug fixes and hardening** — making install, onboarding, and daily use bulletproof
- **Feature development** — executing from a prioritized V2 roadmap (memory pipelines, integrations, team features)
- **Code quality** — tests (435 and growing), CI/CD, documentation
- **OpenClaw compatibility** — keeping the plugin stable across upstream releases

## Tech Stack

- TypeScript (Node.js)
- Lit (web components) for UI
- OpenClaw plugin SDK
- QMD (full-text search / SQLite)
- Honcho (conversational memory)
- Git-based workspace sync
- LLM APIs (Anthropic Claude, OpenAI, xAI)
- MCP (Model Context Protocol) server
