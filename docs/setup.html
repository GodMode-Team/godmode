Welcome to GodMode

You're 6 steps away from your AI-powered second brain. Follow along — this takes about 5 minutes.

1
Install Node.js (v22 LTS)
GodMode runs on Node.js. Grab the latest LTS version, or use the command below.
Download from nodejs.org →  or install via terminal:
$ brew install node@22
> winget install OpenJS.NodeJS.LTS
$ curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash - && \ sudo apt-get install -y nodejs
Verify
$ node --version # should print v22.x.x
  • macOS: Don't have Homebrew? Install it first: /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  • Windows: If winget isn't available, download the installer directly from nodejs.org.
  • Linux: On Fedora/RHEL, use sudo dnf install nodejs. On Arch, use sudo pacman -S nodejs npm.
  • Already have Node but older version? Use nvm to manage multiple versions: nvm install 22 && nvm use 22
2
Install OpenClaw
OpenClaw is the AI gateway that powers GodMode. One command to install it globally.
$ npm install -g openclaw
Verify
$ openclaw --version
  • Permission denied? On macOS/Linux, try sudo npm install -g openclaw or fix your npm prefix: npm config set prefix ~/.npm-global and add ~/.npm-global/bin to your PATH.
  • Command not found after install? Close and reopen your terminal, or run hash -r (bash/zsh) to refresh your PATH.
  • Behind a proxy? Set npm config set proxy http://your-proxy:port before installing.
3
Authenticate with Claude
Connect your Claude account so GodMode can talk to AI models.
Which command should I use?
• openclaw setup-token — Use this if you have a Claude Pro or Max subscription. It links your existing Anthropic account.
• openclaw auth login — Use this if you have a separate API key or want to log in with API credentials.
Option A — Claude Pro / Max subscribers
$ openclaw setup-token
Option B — API key authentication
$ openclaw auth login
  • Browser doesn't open? Copy the URL printed in your terminal and paste it into your browser manually.
  • Token expired? Run the same command again to refresh your credentials.
  • Not sure which plan you have? Check at console.anthropic.com. If you pay monthly for Claude.ai, you likely have Pro or Max.
4
Install the GodMode Plugin
Almost there! This pulls down the GodMode plugin and registers it with OpenClaw.
$ openclaw plugins install @godmode-team/godmode
  • Plugin not found? Make sure you're running OpenClaw v1.0 or later: openclaw --version. Update with npm update -g openclaw.
  • Install hangs? Check your internet connection and try again. If behind a firewall, ensure npm registry access is allowed.
  • Want to install from source? Clone the repo and run openclaw plugins link ./godmode-plugin from the project directory.
5
Activate Your License
Enter your license key to unlock GodMode.
$ openclaw godmode activate GM-...
  • Don't have a key yet? Get one at lifeongodmode.com.
  • "Invalid key" error? Double-check for typos. Keys are case-sensitive and start with GM-.
  • Key already used? Each key is single-use. Contact support at support@lifeongodmode.com if you need to transfer a license.
6
Launch GodMode
Fire up the gateway and open GodMode in your browser. You made it!
$ openclaw gateway start
⚡ Open GodMode
  • Port in use? Another process may be on port 18789. Check with lsof -i :18789 (macOS/Linux) or netstat -ano | findstr :18789 (Windows).
  • Page won't load? Wait a few seconds after starting the gateway — it may need a moment to initialize. Then refresh.
  • Want to run in the background? Use openclaw gateway start --daemon to run the gateway as a background service.
  • Need to restart? Run openclaw gateway restart to restart cleanly.
